Frontend (React + Tailwind) - Quick Start

1) Install dependencies

   In PowerShell (from the project folder):

   cd "h:\Smart Waste Management System1\smart-waste-management\Frontend\smart-waste-management-system"
   npm install

2) Run development server

   npm run dev

3) Notes

- Theme toggling (light/dark) is available in the top-right. The theme persists in localStorage.
- Sidebar can be collapsed using the sidebar toggle in the topbar.
- Pages included: Dashboard (with a sample chart), Reports, About, Services, Contact.
- Tailwind is configured (tailwind.config.cjs). If you change styles, restart the dev server if needed.
